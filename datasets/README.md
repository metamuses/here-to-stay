# Datasets

#### Enrolled students by residence
###### Description
Numero di studenti iscritti per provincia di residenza, provincia della sede didattica del corso di studio, ateneo e gruppo disciplinare del corso

###### Source
[Ustat - Ministero dell'Università e della Ricerca (MUR)](https://dati-ustat.mur.gov.it/dataset/iscritti/resource/b270ef1a-c219-48b1-8399-b1458e225d39)

###### File
[`14a_iscrittixresidenzasedecorsogruppo.csv`](14a_iscrittixresidenzasedecorsogruppo.csv)


#### List of Tourist Accommodation Businesses
